
### Running Locally

1.  **Clone or download** the repository.
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Configure environment variables:**
    Rename `.env.example` to `.env.local` and add your `GEMINI_API_KEY`.
4.  **Run the app:**
    ```bash
    npm run dev
    ```
5.  **Open in browser:**
    Navigate to `http://localhost:5173` to view the app.
