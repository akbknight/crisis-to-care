# Crisis to Care

**Empowering first-generation college students to navigate university life with confidence and support.**

Crisis to Care is a mission-driven web application designed specifically for first-generation students on large university campuses. It provides a safe, anonymous space to connect with peers, find essential campus resources, and handle moments of crisis with grounding and support.

## üåü Why This Exists

First-gen students often face unique challenges: navigating complex administrative systems, balancing work/family with academics, and dealing with a sense of isolation in large institutional environments. This project aims to bridge that gap by providing a central hub for support that understands the first-gen experience.

## üöÄ Features

-   **Anonymous Community:** A place to share stories and connect with other first-gen students without the pressure of identity.
-   **Campus Navigator:** Step-by-step guidance to find the right office or resource on your specific campus.
-   **Crisis Support:** Immediate grounding exercises and a clear path to professional help when things get overwhelming.

## üíª Getting Started

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or higher recommended)
-   A [Gemini API Key](https://aistudio.google.com/app/apikey) (for the AI-powered navigator)

### Running Locally

1.  **Clone or download** the repository.
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Configure environment variables:**
    Rename `.env.example` to `.env.local` and add your `GEMINI_API_KEY`.
4.  **Run the app:**
    ```bash
    npm run dev
    ```
5.  **Open in browser:**
    Navigate to `http://localhost:5173` to view the app.
